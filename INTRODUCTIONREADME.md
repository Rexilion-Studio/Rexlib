Rexlib is a lua library created to save time,
To learn how to use the functions of rexlib check the read me inside the Rexlib folder
Rexlib is free, if you wish you can donate which is fully optional, but is very appreciated
To use, download the Rexlib file, use the install.bat, then you will get  rexlib.lua in a folder called.rexlib in your user section in files,from there you can add it into your terminal or other coding enviroments, it is basicly a lua module, that can be added so you do not have to create your own functions and instead you can use the ones in rexlib which save you some time and allows you to skip the tedious process of creating those functions yourself
